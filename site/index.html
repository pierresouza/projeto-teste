O Onion Mini é a versão mini e portátil do Sistema Onion: desenvolvimento Spec-as-Code — especificação antes do código — em um master-prompt que roda em web chats gratuitos (Claude.ai, ChatGPT, Gemini) e nativamente em IDEs agênticas (Claude Code, Cursor, Codex, Copilot, Zed...). O começo ideal para quem está iniciando no desenvolvimento com IA.

O que vem dentro

Pequeno no tamanho.
Inteiro na disciplina.

🥇
Regra de Ouro
Nunca gerar código antes da especificação: @product especifica o negócio, @engineer planeja a técnica — só então codificar. Com guardião anti-bypass embutido.
🔄
Planejar → Executar → Avaliar
Todo trabalho roda esse ciclo — e a avaliação fecha sempre com redesenho: "o que faremos diferente no próximo?". Disciplina de autorregulação embutida no fluxo, sem cerimônia.
📋
Task Manager Lite
Gestão de tasks agnóstica de ferramenta: o backlog do contexto É o quadro (A Fazer → Em Dev → Feito). Quer Jira, ClickUp ou Trello? Troca só o adapter de prompts.
📚
2 Contextos SSOT
Negócio e Técnica em arquivos únicos e estruturados — visão, personas, dores, métricas, backlog, ADR-lite, débitos. Cada seção mapeia uma camada do Onion completo: graduar é expandir, sem retrabalho.
🤖
AGENTS.md universal
Config no padrão universal lido nativamente por Codex, Cursor, Copilot, Zed e Aider — copiou para a raiz, o agente se auto-configura. Claude Code entra pelo CLAUDE.md incluso.
🥣
Dogfood Mini
Nada está "pronto" sem rodar ou verificar de verdade — conclusão sem evidência é hipótese, e é declarada como tal. A honestidade epistêmica do Onion, de bolso.

Como instalar

Dois cenários. Zero fricção.

💬
Cenário A — Web Chats (grátis)
Claude.ai, ChatGPT ou Gemini: cole o ONION-MASTER-PROMPT.md nas instruções do seu Project/GPT/Gem e anexe os 3 arquivos de docs/. A IA vira o motor lógico; você salva os arquivos.
⚙️
Cenário B — IDEs Agênticas
Copie AGENTS.md + CLAUDE.md + ONION-MASTER-PROMPT.md + docs/ para a raiz do projeto. Pronto: Codex, Cursor, Copilot, Zed e Claude Code auto-descobrem e passam a editar seus arquivos direto.
