# Contexto Inicial da Sessão

## Motivação
O usuário desejava uma ferramenta simples, limpa e moderna que permitisse gerenciar os relacionamentos do Instagram de forma a limpar quem ele segue. A principal dor era a falta de uma ferramenta rápida (estilo cartões/fila) que também fosse segura (aplicativos que realizam login automático e automação direta de APIs no Instagram resultam em banimento severo da conta).

## Posição de Partida
O projeto Onion Mini estava apenas com sua estrutura base de arquivos informativos (site/index.html) e templates de contexto vazios. Não havia código ou estrutura para a ferramenta solicitada.

## Restrições Técnicas & Requisitos
- **Stack:** Next.js 14+ (App Router), JavaScript (ES6+), CSS Modules e CSS puro (conforme preferência dos guidelines de Vanilla CSS).
- **Segurança:** Não coletar credenciais de acesso, sem banco de dados backend, processamento executado 100% no cliente.
- **Portabilidade:** SPA portátil local executada via servidor local simples.
- **Aesthetics:** Design de alta fidelidade visual (luxuoso, cores ricas, Dark Mode automático, interações dinâmicas).
