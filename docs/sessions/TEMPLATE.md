# Session Template — Onion Mini 🧅

Use estes templates para gerar a documentação em cada nova sessão de desenvolvimento (`docs/sessions/YYYY-MM-DD_HHMM_nome-do-topico/`).

---

## 📄 README.md
```markdown
# Nome do Tópico — YYYY-MM-DD

## 🎯 Objetivo
[Descrição do objetivo da sessão]

## 📊 Resultados
- [Lista de entregas / Arquivos criados ou modificados]

## 🔗 Links Relacionados
- [Documentação relacionada, issues ou pull requests]

## ⏱️ Tempo Investido
[Duração aproximada da sessão, ex: 1 hora]
```

---

## 📄 context.md
```markdown
# Contexto — Nome do Tópico

## Situação Inicial
[Estado do projeto antes do início desta sessão]

## Motivação
[Por que este trabalho foi necessário e qual problema resolve]

## Restrições
[Limitações técnicas, de arquitetura, tempo ou recursos]

## Referências
[Links de documentações, discussões ou referências usadas]
```

---

## 📄 decisions.md
```markdown
# Decisões Tomadas — Nome do Tópico

## Decisão 1: [Título da Decisão]
- **Contexto**: [Por que esta decisão foi necessária]
- **Opções Consideradas**:
  - Opção A: [Prós/Contras]
  - Opção B: [Prós/Contras]
- **Decisão**: [Opção escolhida]
- **Justificativa**: [Razão principal da escolha]
- **Impacto**: [Consequências e dependências criadas]
```

```markdown
- [Lista de testes unitários, de integração ou e2e implementados]
```
